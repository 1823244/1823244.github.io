Returns whether all elements of this stream match the provided predicate.

          "java.util.Stream::allMatch: At least one project has no name!");

static <T> Stream.Builder<T> builder()

Returns a builder for a Stream.
This allows the creation of a Stream by generating elements individually and adding them to the Builder (without the copying overhead that comes from using an ArrayList as a temporary buffer.)
